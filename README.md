# Automated Web Data Scraper

This project is part of the **AGNEYRA Python Programming Internship **.

The program automatically extracts data from a website and saves the collected data into a **CSV file**. The script can also run at scheduled intervals for automation.

---

## Project Objective

The goal of this project is to:

- Extract information from a website
- Store the extracted data in a CSV file
- Automate the scraping process
- Demonstrate practical use of Python for data extraction

---

## Features

- Web data extraction using Python
- HTML parsing with BeautifulSoup
- Data storage in CSV format
- Automated scheduled execution
- Simple and beginner-friendly implementation

---

## Technologies Used

- Python 3
- Requests
- BeautifulSoup4
- CSV module
- Schedule library
- VS Code
